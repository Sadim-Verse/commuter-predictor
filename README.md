# Commuter Predictor

A Python micro-project analyzing commute times between Urfahr and Voestalpine in Linz, comparing car vs. public transport.

## Features
- Predicts travel time differences (7:00 AM vs 9:00 AM)
- Visualizes routes on a map
- Uses Austrian open data sources

## Technologies
- Python 3.x
- Jupyter Notebook
- OpenStreetMap API
- Folium for visualization
